Не каждая задача по программированию стоит расхода квоты самых дорогих моделей. FreeUltraCode объединяет Claude Code, Codex, Gemini, бесплатные каналы и локальные модели в одном локальном чате. Дешевые модели подходят для разведки и рутины, а более стабильные можно оставить для важных решений.

## Зачем нужен FreeUltraCode

Coding agents полезны, но квота премиальных моделей быстро заканчивается. FreeUltraCode сохраняет локальный чат и позволяет направлять запросы через бесплатные, пробные или недорогие каналы, когда их достаточно.

- Используйте GitHub Models, Hugging Face Router, SambaNova Cloud, Together AI, Gemini, DeepSeek, Kimi, Groq, OpenRouter, NVIDIA NIM, Z.ai, Kilo, LLM7, Ollama, LM Studio и llama.cpp.
- API-ключи и настройки провайдеров остаются на вашей машине.
- Runtime, канал, режим разрешений и workspace меняются прямо из области ввода.
- История чатов, избранное, запланированные prompts и контекст workspace хранятся локально.
- Локальные модели можно использовать без API-ключей, если хватает оборудования.

## Возможности

### Чат для программирования

- Запрашивайте правки кода, расследование багов, рефакторинг, тесты, release notes или документацию.
- Добавляйте пути к файлам или перетаскивайте файлы в область ввода.
- Смотрите потоковые ответы, логи команд, ссылки на файлы и резюме в одном чате.
- Продолжайте работу уточняющими запросами в той же сессии.

### Генерация изображений + программирование

- Используйте модель генерации изображений и модель для программирования в одной локальной беседе.
- Включайте режим изображения для визуальных ассетов, иконок, постеров и дизайн-референсов, затем возвращайтесь к модели программирования, чтобы применить их в проекте.
- Сгенерированное изображение, prompt, данные провайдера, логи и последующие изменения кода остаются в одной истории.

### Маршрутизация бесплатных моделей

- **20+ удаленных каналов и локальные runtimes**: NVIDIA NIM, OpenRouter, GitHub Models, Hugging Face Router, SambaNova Cloud, Together AI, Google Gemini, DeepSeek, Mistral, Mistral Codestral, OpenCode, Wafer, Kimi, Cerebras, Groq, Fireworks, Z.ai, LLM7, Kilo Gateway, плюс Ollama, LM Studio и llama.cpp.
- **Экспериментальные маршруты без ключа**: LLM7 и Kilo Gateway можно попробовать без API-ключа, но только для несекретных coding prompts.
- **Официальные бесплатные или пробные квоты**: ключи провайдеров хранятся локально в приложении.
- Локальный Rust proxy переводит протоколы Anthropic и OpenAI-compatible.
- Claude Code может работать через настроенные бесплатные каналы без изменения интерфейса чата.
- Ключи, переопределения моделей и локальные модели настраиваются в settings.

### Динамический рабочий процесс (/ultracode)

Для сложных многоэтапных задач программирования `/ultracode <задача>` мгновенно генерирует специализированный исполняемый harness и сразу запускает его. Визуальный холст не требуется.

- Опишите задачу на естественном языке — планировщик строит harness с параллельными подагентами, состязательной проверкой и воротами приёмки.
- Шесть внутренних стратегий выбираются автоматически: классифицировать-и-действовать, веер-и-синтез, состязательная-проверка, генерировать-и-фильтровать, турнир, цикл-до-завершения.
- Каждый запуск полностью протоколируется в `.fuc-run/<run-id>/` с реестром задач, событиями, вердиктом и итоговым результатом.
- Запуск из десктоп-приложения или через CLI: `fuc ultracode "<задача>" --json --interactive --cwd <workspace>`.
- Нулевая конфигурация — используются локальные учётные данные `claude` CLI.

#### Free Auto — Автоматическое переключение каналов

Канал **Auto** (`freecc:auto` в меню Channel) автоматически направляет каждый запрос через лучший доступный бесплатный канал без ручного переключения.

- Перебирает все настроенные бесплатные каналы, автоматически пропуская те, что достигли ограничений (429) или возвращают ошибки upstream (5xx).
- Отслеживает время охлаждения каждого канала с откатом: после ошибки канал приостанавливается перед повторной попыткой.
- Поддерживает опциональную перезапись модели, чтобы все авто-маршрутизируемые запросы использовали одну модель.
- Если все каналы исчерпаны, возвращает 503 с журналом ошибок для диагностики.

#### Цепочка поставщиков: DeepSeek → CodeX

С `/ultracode` harness может автоматически связывать нескольких поставщиков между шагами плана. Типичный сценарий: DeepSeek создаёт быстрые черновики с низкими затратами, CodeX дорабатывает результат до финального качества.

- **Динамический план harness** поддерживает перезапись `model` для каждого шага — назначьте DeepSeek для мозгового штурма/классификации, а CodeX/Gemini для реализации/проверки.
- **Совместимость с cc-switch**: FreeUltraCode читает конфигурацию `cc-switch` CLI; любой поставщик, уже настроенный для Claude Code, сразу доступен для шагов ultracode.
- Стратегия **веер-и-синтез** распараллеливает DeepSeek-воркеров по независимым подзадачам, затем консенсус-гейт (CodeX) синтезирует и проверяет результаты.

#### Выбор канала с учётом скорости

Auto-канал бесплатного прокси приоритизирует каналы на основе сигналов реальной доступности:

- **Учёт ограничений скорости**: каналы с 429 охлаждаются на 30+ секунд перед повтором, избегая бесполезных попыток.
- **Быстрый отказ при ошибках**: неповторяемые ошибки (4xx аутентификация, 5xx падение upstream) отслеживаются с охлаждением; Auto-роутер их пропускает.
- **Бюджет времени соединения**: каждая попытка ограничена таймаутом upstream; Auto-роутер не блокируется на одном медленном upstream.
- **Естественный порядок по отзывчивости**: успешные каналы пробуются первыми; ошибочные каналы отодвигаются в конец списка.

Эти возможности обеспечивают устойчивость запусков `/ultracode`, даже когда отдельные бесплатные поставщики медленны, ограничены или временно недоступны.

### Использование режима изображения

Режим изображения превращает поле чата в ввод для text-to-image, сохраняя ту же историю сессии. Он полезен для UI-ассетов, иконок, постеров и дизайн-референсов перед возвратом к коду.

1. Откройте **Settings** -> **Images**, выберите провайдера изображений по умолчанию и заполните API key, Account ID, Base URL или локальный endpoint ComfyUI.
2. В сессии чата введите `/image-mode-start`. Можно сразу добавить первый prompt:

```text
/image-mode-start чистая иконка приложения для локального coding agent, стеклянный эффект, 1024x1024
```

3. Пока режим включен, обычные сообщения генерируют изображения вместо правок кода. Селектор **Channel** переключается на провайдеров изображений.
4. Опишите нужное изображение. FreeUltraCode сначала улучшает prompt через модель программирования, затем отправляет его настроенному провайдеру.

<p align="center">
  <img src="images/生图/image-mode-session.ru.png" alt="Режим изображения генерирует изображения в той же сессии FreeUltraCode" width="720">
</p>

5. Отправьте `/image-mode-end`, чтобы вернуться к каналу и модели программирования. Для одной картинки без постоянного режима используйте `/image`, `/img`, `/draw`, `/生图` или `/画图` с prompt.

## Как это работает

```text
Запрос пользователя
    |
    v
Chat composer
    |
    +--> выбранные runtime / канал / разрешения / workspace
             |
             +--> API провайдера, локальный CLI или локальный free-channel proxy
                        |
                        +--> потоковый вывод, лог инструментов и история чата
```
